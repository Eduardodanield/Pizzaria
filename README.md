
# 🍕 **Pizzaria TT — Seu Pedido Online em Minutos!**

![Banner da Pizzaria TT](https://i.imgur.com/yw1a1U9.png)
*Sistema de pedidos online desenvolvido como projeto acadêmico.*

---

## 🍽️ Sobre o Projeto

A **Pizzaria TT** é uma loja virtual criada para simular o funcionamento de um sistema de pedidos online.
O projeto foi desenvolvido como parte de um trabalho acadêmico, com foco em **interface, usabilidade e experiência do cliente**.

O objetivo é permitir que o usuário explore o cardápio, escolha sabores e tamanhos, monte seu pedido e visualize o total de forma simples e intuitiva.

---

## 🧠 Tecnologias Utilizadas

* **HTML5** — estrutura principal
* **CSS3 / Tailwind CSS** — design moderno e responsivo
* **JavaScript (puro)** — lógica de interação e controle de pedidos
* **Node.js / NPM** — gerenciamento de dependências e ambiente local
* **Vite** (opcional) — para desenvolvimento rápido

---

## ⚙️ Funcionalidades

✅ Exibição de cardápio com sabores e tamanhos
✅ Adição e remoção de itens do carrinho
✅ Cálculo automático do valor total
✅ Interface leve e responsiva
✅ Simulação de pedido online

## 🚀 Como Executar Localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/Eduardodanield/Pizzaria.git
   ```

2. Acesse a pasta:

   ```bash
   cd Pizzaria
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Execute o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

5. Abra no navegador:

   ```
   http://localhost:xxxx
   ```
